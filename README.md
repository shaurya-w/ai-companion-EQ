# EQ Lens

**EQ lens** is a small, intentional space for reflection.

It’s designed to feel calm, private, and human. The AI is trained so as to assisting users to reflect and make sense of their inner world. 

### Product Features
- Guided reflection and supportive conversation
- Calm, non-judgmental tone. Help the user reflect without spiraling
- Session-only interactions — nothing is stored
- Authenticated access
- Clear disclaimers: not a therapist or medical tool
- Safe, private experience designed to build trust

### Developer Features
- **Next.js 16 App Router** for modern server-rendered routes
- **Serverless API routes** for dynamic AI responses
- **Vercel-ready deployment** — zero configuration
- **Gemini (OpenAI-compatible API)** integration
- **Streaming response-ready** (for smooth UX, optional)
- **Planned rate limiting** to prevent abuse
- **Firebase Auth integration** for access control
- No database (by design)
